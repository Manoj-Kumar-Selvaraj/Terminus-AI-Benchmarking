Exactly two open cycle days is still eligible; three or more is not. Dates missing from the calendar are not open, and a blank settlement date or blank return date must be treated as ineligible rather than as a same-day match. If more than one unused voucher satisfies all matching and cycle-window rules for a return, clear against the eligible voucher with the latest settlement date; when dates tie, use the earliest voucher input row.
